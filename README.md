# Игра в жанре RogueLike

RogueLike, разрабатываемый в рамках курса "ООП и Гибкая методология разработки" в МПГУ ИМО.

## Концепция

Несложный роглайк с собиранием монеток и уничтожением врагов на пути.

*Другие идеи для игры еще в разработке.*

## Управление

Простое управление на WASD или на стрелочки.

*Возможны изменения.*

## Архитектура

### Используемые паттерны проектирования

| Паттерн | Где используется | Что делает |
|---------|---------------|------------|
| **Singleton** | `GameManager` | Гарантирует, что менеджер игры существует в единственном экземпляре |
| **Factory Method** | `EnemyFactory`, `GoblinFactory`, `SkeletonFactory` | Создание врагов разных типов через общий интерфейс |
| **Prototype** | `Enemy.Clone()` | Копирование врагов для быстрого создания групп |
| **Decorator** | `PoisonDecorator`, `FireDecorator`, `RustDecorator` | Добавление эффектов к оружию без изменения его кода |
| **Adapter** | `LoggerAdapter` | Приведение интерфейса внешнего логгера к единому стандарту |
